FAQ

Frequently Asked Questions

Does it work with Odoo Community Edition?
Yes. Depends only on stock and product, both included in CE. No Enterprise features required.
Does it support multiple warehouses?
Yes. Select one or multiple warehouses/locations. The report will aggregate or separate as needed.
Is the valuation FIFO or average?
Uses the unit cost from each stock move (price_unit) and falls back to the product’s standard_price. Works with any costing method.
Can I use this in Odoo Online / Odoo.sh?
Yes. Odoo.sh supports custom modules. For Odoo Online (SaaS), third-party Python modules may have limited support.
