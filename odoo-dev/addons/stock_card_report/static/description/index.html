Enhanced Stock Card Report

Multi-warehouse stock movement report with running balance, valuation, and export.

For Odoo Community Edition 18.0 & 19.0

100% Free & Open Source — AGPL-3 licensed. No dependencies on Odoo Enterprise or paid modules.

Features

  • Filter by date range, product, category, warehouse, or location
  • Running balance with unit cost valuation (incoming/outgoing value)
  • Interactive bar chart (incoming vs outgoing over time)
  • Pivot table by product/month
  • Printable PDF report per product
  • Export to Excel via standard Odoo export
  • Opening balance before date range included

Screenshots

Usage

  1. Go to Inventory > Reporting > Stock Card Report
  2. Select date range, products/warehouses (optional)
  3. Click Generate to view the report
  4. Use Print PDF for a per-product printable report
  5. Switch between List, Graph, Pivot views for different analysis

Requirements

  • Odoo Community Edition 18.0 or 19.0
  • stock, product modules (included in CE)
